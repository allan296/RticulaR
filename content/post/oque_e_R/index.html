O que é o R?

O R é uma linguagem e um ambiente que disponibiliza uma grande variedade de técnicas estatísticas (modelos lineares e não lineares, testes estatítsicos clássicos, análises temporais, entre outros) e gráficas. Existem diversos motivos para você utilizá-lo, vou citar os 5 principais: Gratuidade, alta popularidade, alto poder e grande suporte.

1) É e sempre será gratuito :heart:

Projeto GNU logo

O R é gratuito para instalar, usar, atualizar, modificar e distribuir e estudar. Diferentemente de outros (e.g. SAS, SPSS), esse software faz parte de um projeto GNU, isso significa que o R respeita a liberdade do usuário, segundo o projeto, softwares pagos colocam os usuários sobre o poder dos desenvolvedores desses softwares.

{{% alert note %}} Um software “livre” não necessáriamente é gratuito. O termo livre, refere-se à liberdade, não ao preço. Ou seja, você tem acesso ao código fonte do software e pode modificá-lo como quiser. {{% /alert %}}
